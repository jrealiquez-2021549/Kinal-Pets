README
Nombre: Veterinaria2025BM

Descripción:
Este es un sistema de gestión para veterinarias desarrollado en Java utilizando JavaFX para la interfaz gráfica. El programa permite administrar clientes, mascotas, citas, consultas, empleados, proveedores, medicamentos, vacunas, facturación e informes. El sistema incluye un módulo de autenticación para que los usuarios puedan iniciar sesión y acceder a las funcionalidades según sus permisos.

Características:
- Autenticación de Usuarios: Permite el inicio y cierre de sesión, validando credenciales contra una base de datos.
- Gestión de Clientes y Mascotas: Registro, consulta y actualización de datos personales y veterinarios.
- Agenda de Citas y Consultas: Programación de citas, registro de consultas médicas y seguimiento clínico.
- Inventario y Compras: Control de medicamentos, tratamientos, proveedores y compras realizadas.
- Vacunaciones: Registro y control de vacunas aplicadas a las mascotas.
- Facturación e Informes: Emisión de facturas detalladas y generación de reportes para administración.
- Interfaz Gráfica Moderna: Implementada con JavaFX y archivos FXML para una separación clara entre lógica y presentación.
- Manejo de Errores y Validaciones: Validaciones exhaustivas en formularios y mensajes claros para el usuario.
- Recursos Multimedia: Uso de imágenes, íconos y sonidos para mejorar la experiencia de usuario.

Requisitos:
- Java 8 o superior: El sistema está desarrollado con Java 8 y es compatible con versiones posteriores.
- JavaFX: Para la ejecución de la interfaz gráfica.
- IDE (NetBeans, IntelliJ IDEA, u otro): Recomendado para compilar, ejecutar y modificar el proyecto.
- Base de datos SQL: Configurada según el script incluido en el proyecto.

Cómo ejecutar:
- Abrir el proyecto en el IDE compatible.
- Configurar la conexión a la base de datos según sea necesario.
- Compilar el proyecto completo.
- Ejecutar la clase principal Principal.java.
- Aparecerá la ventana de inicio de sesión, donde se ingresan las credenciales para acceder al sistema.
- Navegar y utilizar las funcionalidades disponibles en el menú principal según el usuario.
